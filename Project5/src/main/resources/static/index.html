欺诈识别模型说明

🔴 盗刷识别模型

基于多维度特征识别被盗卡片交易，包括：大额异常消费、深夜时段交易、异地交易、 新设备首次交易、陌生IP地址、短时间密集交易、高变现商品类别等特征。

置信度阈值: 70分 模型版本: v2.0

🟡 套现识别模型

识别信用卡套现行为，特征包括：整数额度消费、当日大额笔数异常、 高额度使用率、疑似套现商户类别、短时间多笔消费、接近信用额度等。

置信度阈值: 65分 模型版本: v2.0

🟡 虚假交易识别

识别虚假/测试交易，特征包括：极低金额交易、验证性小额金额、 收付账户相同、异常交易描述、可疑商户名称、缺少设备/IP信息等。

置信度阈值: 70分 模型版本: v2.0

🟣 账户盗用模型

识别账户被劫持/盗用风险，特征包括：资金转移类交易、大额转出余额、 IP与登录不一致、全新设备操作、位置突变、转账至新收款人等。

置信度阈值: 65分 模型版本: v2.0

溯源功能说明

交易溯源系统提供完整的交易全生命周期追踪能力，支持：

  • 决策链路追踪：完整记录交易从提交到最终决策的每一步骤
  • 规则触发历史：详细记录每条风控规则的触发情况和评分
  • 模型推理记录：机器学习各模型的推理结果和置信度分数
  • 证据链管理：自动收集和汇总欺诈检测的各类证据
  • 关联交易/告警：自动关联相关交易和风险告警
  • 人工干预记录：完整记录人工审核操作和决策原因
  • 审计追踪：所有操作均可追溯，满足监管合规要求
